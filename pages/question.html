Question-1: What are the different ways to select an element in the DOM?

Ans: There are someways in which can select elements in a DOM using selectors. getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

Ans: textContents is an element that returns all the text contained within all child elements. innerText is an element that returns all the text contained within all child elements. and innerHtml returns all the text contained within an element, including the html tags.

Question-3: What is event delegation in the DOM?

Ans: Event delegation is a JavaScript technique that allows an event listener to be attached to a parent element instead of attaching event listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

Ans: Bubbling in the DOM is an event propagation method. It starts with an event triggering on the innermost element and then proceeds to trigger the same event on its parent element and then its parent's parent element and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans: To create an element in JavaScript, use document.createElement(). To add it to a parent element, use appendChild(), append() methods. To remove an element, use removeChild(), remove().
